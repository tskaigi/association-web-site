## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## デプロイ手順

main ブランチにマージされると、自動でビルドして更新されます。
ブランチを切り、PR を出してください。マージされると自動でデプロイされます。


## 情報の追加方法

### イベント情報

[page.tsx](src/app/page.tsx)の、conferences変数を修正する

### 理事情報

[page.tsx](src/app/page.tsx)の、members変数を修正する

### 電子公示情報

[page.tsx](src/app/page.tsx)の、reports変数を修正する
